Algebraic Logic Functional programming language

Algebraic Logic Functional (ALF) est un langage de programmation basé sur la logique algébrique et fonctionnelle. Il combine des concepts de programmation logique avec des fonctions d'ordre supérieur et des types de données algébriques, ce qui permet une programmation déclarative et expressive.

Exemple de code ALF :

sort list : list of int {
            nil {
              []
            } | {
              insert(x, sort(tail))
            }
          }
          
          insert x:list of int, y:list of int : list of int {
            nil {
              [x]
            } | {
              if x<=head then
                [x,y]
              else
                [head, insert(x,tail)]
            }
          }

Algol 60

Algol 60 est un langage de programmation impératif qui a été développé dans les années 1960. Il a joué un rôle important dans le développement des langages de programmation modernes et a introduit de nombreuses fonctionnalités clés, telles que les blocs structurés, les variables locales statiques, la récursivité et la portée lexicale.

Exemple de code Algol 60 :

PROCEDURE factor(n);
              BEGIN INTEGER i;
                FOR i := 2 STEP 1 UNTIL n DIV 2 DO
                  IF n MOD i = 0 THEN
                    BEGIN print(i); factor(n DIV i); RETURN END
                  END
                END
              END;
              BEGIN INTEGER n;
                read(n); factor(n); print(n)
              END.

Algol 68

Algol 68 est un langage de programmation impératif, orienté objet et fonctionnel qui a été conçu pour succéder à Algol 60. Il propose de nombreuses fonctionnalités avancées, notamment des types de données complexes, des structures flexibles et des opérations sur les listes et les tableaux.

Exemple de code Algol 68 :

PROC comb := (INT n, INT k)VOID:
              (
                INT i, c;
                IF k = 0 | n = k THEN
                  c := 1
                ELSE
                  c := comb(n - 1, k - 1) + comb(n - 1, k)
                FI;
                print(c)
              );
              comb(5, 2)

Algol W

Algol W est une version étendue du langage Algol 60 qui a été développée dans les années 1980. Elle a introduit des fonctionnalités supplémentaires telles que la gestion des exceptions, les expressions rationnelles et les types de données abstraits, ce qui en fait un langage plus puissant et plus expressif.

Exemple de code Algol W :

            WHILE INPUT > 0 DO
              WRITE(INPUT MOD 10); INPUT := INPUT DIV 10
            END
          END Reverse;
          
          VAR n: INTEGER;
          
          BEGIN
            READ(n); Reverse(n)
          END.

Microsoft Small Basic

Microsoft Small Basic est un langage de programmation simplifié conçu pour les débutants. Il offre une interface conviviale et des fonctionnalités de base pour apprendre les concepts de la programmation.

TextWindow.WriteLine("Bonjour, monde !")

PowerBasic

PowerBasic est un langage de programmation compatible avec le langage BASIC. Il offre des fonctionnalités avancées et une meilleure performance par rapport à d'autres dialectes de BASIC.

Exemple de code PowerBasic :

PRINT "Bonjour, monde !"
          SLEEP
