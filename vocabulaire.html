Def'Web

Vocabulaire

A

a

La balise <a> définit un lien hypertexte. Elle est utilisée pour créer des liens vers d'autres pages, des fichiers ou des emplacements dans la même page.

Exemple de balise <a> :

<a href="https://example.com">Lien vers exemple.com</a>

abbr

La balise <abbr> définit une abréviation ou un acronyme. Elle est utilisée pour indiquer la signification complète d'un mot ou d'une expression lorsqu'il est survolé par la souris.

Exemple de balise <abbr> :

<abbr title="Hypertext Markup Language">HTML</abbr>

address

La balise <address> définit les informations de contact pour l'auteur ou le propriétaire d'un document ou d'une section. Elle peut contenir une adresse postale, un numéro de téléphone, une adresse e-mail, etc.

Exemple de balise <address> :

<address>
          John Doe
          123 Rue des Exemples
          12345 Ville
        </address>

article

La balise <article> est utilisée pour représenter un contenu indépendant et autonome dans un document HTML. Elle peut être utilisée pour encapsuler des articles de blog, des commentaires, des widgets, etc. Elle devrait avoir un sens complet en étant extraite de son contexte.

Exemple de balise article :

<article>
          <h2>Titre de l'article</h2>
          <p>Contenu de l'article...</p>
        </article>

aside

La balise <aside> est utilisée pour représenter un contenu connexe ou supplémentaire qui est lié au contenu principal du document, mais qui peut être considéré comme distinct. Elle peut être utilisée pour afficher des encarts publicitaires, des citations, des biographies, etc.

Exemple de balise aside :

<aside>
          <h3>À propos de l'auteur</h3>
          <p>Informations supplémentaires...</p>
        </aside>

B

b

La balise <b> est utilisée pour mettre en évidence un texte important dans un document HTML. Par défaut, le texte entouré de balises <b> est affiché en gras, mais son utilisation actuelle est plus générale et dépend de la signification donnée au texte.

Exemple de balise b :

<b>Texte important</b>

Balise

En HTML, une balise est un élément utilisé pour marquer et structurer le contenu d'une page web. Elle est composée d'une balise ouvrante (<span>) et d'une balise fermante (</span>). Les balises définissent la structure et le sens du contenu.

Exemple de balise HTML :

<h1>Titre</h1>

base

La balise `base` définit l'URL de base pour les liens d'une page HTML. Elle est utilisée pour spécifier l'URL de base à partir de laquelle tous les liens relatifs dans le document seront résolus.

Exemple de balise `base` :

<head>
          <base href="https://www.example.com/">
        </head>

bdi

La balise <bdi> est utilisée pour isoler une partie de texte qui peut être formatée différemment en fonction de la direction du texte environnant. Elle est souvent utilisée pour gérer correctement l'affichage de texte dans des langues qui s'écrivent de droite à gauche (comme l'arabe ou l'hébreu) lorsqu'il est mélangé avec du texte de gauche à droite.

Exemple de balise bdi :

<bdi>العالم</bdi>

bdo

La balise <bdo> est utilisée pour spécifier explicitement la direction d'un texte dans un document HTML. Elle est souvent utilisée pour formater correctement le texte dans des langues qui s'écrivent de droite à gauche (comme l'arabe ou l'hébreu) ou de gauche à droite (comme l'anglais ou le français).

Exemple de balise bdo :

<bdo dir="rtl">العالم</bdo>

blockquote

La balise <blockquote> est utilisée pour citer un contenu provenant d'une autre source. Elle indique que le texte cité est une citation et peut être utilisée pour mettre en valeur ou styliser le texte cité.

Exemple de balise blockquote :

<blockquote>
          Voici une citation.
        </blockquote>

body

La balise `body` représente le contenu principal d'une page HTML. Elle contient tous les éléments visibles à l'écran, tels que le texte, les images, les liens, etc.

Exemple de balise `body` :

<body>
          <h1>Titre de la page</h1>
          <p>Contenu de la page...</p>
        </body>

br

La balise <br> est utilisée pour insérer un saut de ligne (retour à la ligne) à l'intérieur d'un paragraphe ou d'un texte. Elle est généralement utilisée lorsque vous souhaitez créer une nouvelle ligne sans créer un nouveau paragraphe.

Exemple de balise br :

Texte sur une ligne <br> Texte sur une nouvelle ligne

C

cite

La balise <cite> est utilisée pour indiquer une référence à une source dans un document HTML. Elle est généralement utilisée pour citer des livres, des articles, des œuvres d'art, etc., et peut être stylisée pour indiquer visuellement qu'il s'agit d'une citation.

Exemple de balise cite :

<cite>Titre de l'article</cite>

code

La balise <code> est utilisée pour représenter du code informatique dans un document HTML. Elle est généralement utilisée pour afficher du code source, des commandes, des expressions, etc., de manière à ce qu'elles soient distinguées du texte normal et à largeur fixe.

Exemple de balise code :

<code>var x = 5;</code>

D

data

La balise <data> est utilisée pour représenter une valeur numérique ou scientifique dans un document HTML. Elle peut être utilisée pour ajouter des métadonnées aux éléments HTML et pour faciliter la manipulation des données via des scripts.

Exemple de balise data :

<p>La population mondiale est d'environ <data value="7800000000"></data> habitants.</p>

del

La balise <del> est utilisée pour représenter du texte supprimé ou barré dans un document HTML. Elle est souvent utilisée pour indiquer des suppressions, des omissions ou des modifications dans le texte existant, notamment dans le contexte de la comparaison ou de la révision de texte.

Exemple de balise del :

<del>Texte supprimé</del>

details, summary, dialog

Ces balises sont utilisées pour créer des éléments interactifs de l'interface utilisateur, tels que des sections déroulantes (avec les balises details et summary) et des boîtes de dialogue modales (avec la balise dialog).

Exemple de balises interactives :

<details>
          <summary>Titre de la section</summary>
          Contenu de la section
        </details>
        
        <dialog open>
          <p>Contenu de la boîte de dialogue</p>
          <button>Fermer</button>
        </dialog>

dfn

La balise <dfn> est utilisée pour indiquer la définition d'un terme dans un document HTML. Elle est souvent utilisée pour fournir une explication ou un contexte supplémentaire pour un terme qui peut ne pas être familier à tous les lecteurs.

Exemple de balise dfn :

<dfn>Terme à définir</dfn>

div

La balise <div> est utilisée pour créer une division ou une section générique dans un document HTML. Elle est souvent utilisée comme un conteneur pour regrouper d'autres éléments HTML et appliquer des styles ou des fonctionnalités à ces éléments groupés.

Exemple de balise div :

<div>
          <p>Contenu du div</p>
          <img src="image.jpg" alt="Image">
        </div>

dl, dt, dd

Les balises <dl>, <dt> et <dd> sont utilisées pour créer une liste de définitions dans un document HTML. <dl> est le conteneur principal, <dt> est utilisée pour définir le terme ou le nom, et <dd> est utilisée pour définir la description ou la signification du terme.

Exemple de balises dl, dt, dd :

<dl>
          <dt>Terme 1</dt>
          <dd>Description 1</dd>
          <dt>Terme 2</dt>
          <dd>Description 2</dd>
        </dl>

Doctype

Le doctype est une déclaration placée au début d'un document HTML pour indiquer au navigateur quel type de document il est en train de traiter. Il définit également la version de HTML utilisée et permet au navigateur d'interpréter correctement le code HTML.

Exemple de doctype HTML5 :

<!DOCTYPE html>

E

Élément

En HTML, un élément est une partie de la structure d'une page web qui est délimitée par une balise ouvrante et une balise fermante, ou une balise auto-fermante. Les éléments peuvent contenir du texte, des images, d'autres éléments et des attributs qui définissent leurs caractéristiques.

Exemple d'élément HTML :

<p>Ceci est un paragraphe.</p>

strong

La balise <strong> est utilisée pour indiquer un texte important ou fortement mis en évidence dans un document HTML. Par défaut, le texte entouré de balises <strong> est affiché en gras.

Exemple de balise strong :

<strong>Texte important</strong>

F

figure, figcaption

Les balises <figure> et <figcaption> sont utilisées pour encapsuler une image ou un média avec une légende associée dans un document HTML. La balise <figure> est utilisée comme conteneur principal et la balise <figcaption> est utilisée pour définir la légende.

Exemple de balises figure, figcaption :

<figure>
          <img src="image.jpg" alt="Image">
          <figcaption>Légende de l'image</figcaption>
        </figure>

form, input, label, textarea, select, option, optgroup, button, fieldset, legend, datalist, progress, output, meter

Ces balises sont utilisées pour créer des formulaires interactifs dans un document HTML, permettant aux utilisateurs de saisir et de soumettre des données.

Exemple de balises de formulaire :

<form action="/submit" method="post">
          <label for="name">Nom :</label>
          <input type="text" id="name" name="name">
        
          <label for="email">Email :</label>
          <input type="email" id="email" name="email">
        
          <input type="submit" value="Envoyer">
        </form>

footer

La balise <footer> représente le pied de page d'une section ou d'une page. Elle contient généralement des informations de droits d'auteur, des liens vers des pages connexes, des coordonnées de contact, etc.

Exemple de balise footer :

<footer>
          <p>© 2023 Mon Site. Tous droits réservés.</p>
          <nav>
            <a href="#">Accueil</a>
            <a href="#">À propos</a>
            <a href="#">Contact</a>
          </nav>
        </footer>

H

head

La balise <head> est utilisée pour définir l'en-tête d'un document HTML. Elle contient des éléments qui ne sont pas affichés directement dans la page, tels que le titre, les liens vers des feuilles de style, les scripts, les métadonnées, etc.

Exemple de balise head :

<head>
          <title>Titre de la page</title>
          <link rel="stylesheet" href="style.css">
        </head>

h1-h6

Les balises <h1> à <h6> sont utilisées pour définir les titres et sous-titres dans un document HTML. <h1> est le titre le plus important, tandis que <h6> est le moins important. Ces balises fournissent une structure hiérarchique au contenu et ont également un impact sur l'accessibilité et le référencement.

Exemple de balise h1-h6 :

<h1>Titre principal</h1>
        <h2>Sous-titre</h2>
        <h3>Sous-sous-titre</h3>

header

La balise <header> représente l'en-tête d'une section ou d'une page. Elle peut contenir des éléments tels que le logo, le titre, la navigation, etc., généralement situés en haut de la page.

Exemple de balise header :

<header>
          <h1>Mon Site</h1>
          <nav>
            <a href="#">Accueil</a>
            <a href="#">À propos</a>
            <a href="#">Contact</a>
          </nav>
        </header>

hr

La balise <hr> est utilisée pour insérer une ligne horizontale dans un document HTML. Elle est souvent utilisée pour séparer visuellement des sections ou des éléments dans une page.

Exemple de balise hr :

<hr>

html

La balise `html` représente l'élément racine d'un document HTML. Elle englobe tous les autres éléments du document, y compris la balise `head` et la balise `body`.

Exemple de balise `html` :

<!DOCTYPE html>
        <html>
          <head>
            <title>Titre de la page</title>
          </head>
          <body>
            <h1>Contenu de la page</h1>
          </body>
        </html>

I

i

La balise <i> est utilisée pour mettre en évidence un texte dans un document HTML. Traditionnellement, elle était utilisée pour afficher le texte en italique, mais son utilisation actuelle est plus générale et dépend de la signification donnée au texte.

Exemple de balise i :

<i>Texte en italique</i>

img, picture, source, video, audio, track, iframe, object, param, embed, map, area, svg, math

Ces balises sont utilisées pour intégrer et afficher des médias (images, vidéos, fichiers audio) et des éléments interactifs dans un document HTML.

Exemple de balises d'intégration de médias : 

<img src="image.jpg" alt="Image">
        <video src="video.mp4" controls></video>
        <audio src="audio.mp3" controls></audio>

ins

La balise <ins> est utilisée pour représenter du texte inséré dans un document HTML. Elle est souvent utilisée pour indiquer des modifications ou des ajouts apportés au texte existant, notamment dans le contexte de la comparaison ou de la révision de texte.

Exemple de balise ins :

<ins>Texte inséré</ins>

K

kdb

La balise <kdb> est utilisée pour représenter une entrée clavier dans un document HTML. Elle est souvent utilisée pour indiquer les touches à appuyer ou les commandes à saisir dans des tutoriels, des documentations techniques, etc.

Exemple de balise kdb :

<kdb>Ctrl + C</kdb>

L

link

La balise `link` est utilisée pour relier un document HTML à une ressource externe. Elle est couramment utilisée pour lier une feuille de style CSS, une icône de site, une police personnalisée, etc.

Exemple de balise `link` :

<head>
          <link rel="stylesheet" href="styles.css">
        </head>

M

main

La balise <main> représente le contenu principal d'une section ou d'une page. Elle doit être unique et ne doit pas être imbriquée à l'intérieur d'autres éléments tels que <header>, <footer> ou <article>. Le contenu principal d'une page, tel que les articles, les sections ou les éléments centraux, devrait être placé à l'intérieur de la balise <main>.

Exemple de balise main :

<main>
          <h1>Contenu principal</h1>
          <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
        </main>

mark

La balise <mark> est utilisée pour mettre en évidence ou surligner une partie du texte dans un document HTML. Elle est souvent utilisée pour attirer l'attention de l'utilisateur sur un texte spécifique.

Exemple de balise mark :

<mark>Texte surligné</mark>

meta

La balise <meta> est utilisée pour spécifier des métadonnées sur un document HTML. Ces métadonnées fournissent des informations sur la page, telles que l'encodage des caractères, la description, les mots-clés, etc. Les balises <meta> ne sont pas affichées à l'utilisateur, mais sont utilisées par les navigateurs et les moteurs de recherche.

Exemple de balise meta :

<meta charset="UTF-8">

N

nav

La balise <nav> est utilisée pour représenter une section de navigation dans un document HTML. Elle est utilisée pour regrouper les liens de navigation tels que les menus, les listes de navigation, etc. Lorsqu'elle est utilisée correctement, elle permet aux utilisateurs de naviguer plus facilement à travers le site.

Exemple de balise nav :

<nav>
          <ul>
            <li><a href="#accueil">Accueil</a></li>
            <li><a href="#services">Services</a></li>
            <li><a href="#contact">Contact</a></li>
          </ul>
        </nav>

O

ol, ul, li

Les balises <ol>, <ul> et <li> sont utilisées pour créer des listes dans un document HTML. <ol> est utilisée pour une liste ordonnée (numérotée), <ul> est utilisée pour une liste non ordonnée (à puces) et <li> est utilisée pour les éléments de la liste.

Exemple de balises ol, ul, li :

<ol>
          <li>Élément 1</li>
          <li>Élément 2</li>
          <li>Élément 3</li>
        </ol>
        
        <ul>
          <li>Élément A</li>
          <li>Élément B</li>
          <li>Élément C</li>
        </ul>

P

p

La balise <p> est utilisée pour définir un paragraphe de texte dans un document HTML. Elle est couramment utilisée pour séparer et structurer le texte en paragraphes.

Exemple de balise p :

<p>Ceci est un paragraphe.</p>

pre

La balise <pre> est utilisée pour afficher du texte préformaté. Le texte à l'intérieur de la balise <pre> est affiché avec une police à largeur fixe et conserve les espaces, les sauts de ligne et les autres espaces blancs tels qu'ils sont écrits dans le code HTML.

Exemple de balise pre :

<pre>
          Code préformaté
          avec des espaces
          et des sauts de ligne
        </pre>

R

ruby, rb, rt, rtc, rp

Les balises <ruby>, <rb>, <rt>, <rtc> et <rp> sont utilisées pour représenter le texte de rubis, qui est utilisé dans les langues asiatiques pour indiquer la prononciation ou la signification d'un caractère ou d'un mot.

Exemple de balises ruby, rb, rt, rtc, rp : 

<ruby>漢<rt>かん</rt></ruby>

S

s

La balise <s> est utilisée pour indiquer que le texte est barré ou rayé dans un document HTML. Elle peut être utilisée pour indiquer du texte supprimé, des informations obsolètes ou tout autre texte qui n'est plus valide ou pertinent.

Exemple de balise s :

<s>Texte barré</s>

script, canvas, template, noscript, slot

Ces balises sont utilisées pour ajouter des fonctionnalités interactives, du contenu dynamique et du contenu conditionnel à un document HTML.

Exemple de balises pour le contenu dynamique : 

<script src="script.js"></script>
        <canvas id="myCanvas"></canvas>
        <template>Modèle de contenu</template>
        <noscript>Contenu alternatif</noscript>
        <slot name="content">Contenu par défaut</slot>

section

La balise <section> est utilisée pour diviser une page en sections thématiques ou fonctionnelles. Elle aide à organiser et structurer le contenu de manière logique et cohérente. Les sections peuvent contenir des en-têtes, des paragraphes, des listes, des images, etc.

Exemple de balise section :

<section>
          <h2>Titre de la section</h2>
          <p>Contenu de la section...</p>
        </section>

small

La balise <small> est utilisée pour indiquer que le texte est de taille réduite par rapport au texte environnant. Elle est généralement utilisée pour afficher des légendes, des annotations, des notes de bas de page ou tout autre texte qui doit être présenté à une taille de police plus petite.

Exemple de balise small :

<small>Texte de petite taille</small>

span

La balise <span> est utilisée pour grouper des éléments en ligne et leur appliquer des styles spécifiques à l'aide de CSS. Elle n'a aucun effet sémantique par elle-même, mais peut être utilisée comme un conteneur pour appliquer des styles ou des scripts à une partie du texte.

Exemple de balise span :

<span class="highlight">Texte avec style</span>

strong

La balise <strong> est utilisée pour indiquer un texte important ou fortement mis en évidence dans un document HTML. Par défaut, le texte entouré de balises <strong> est affiché en gras.

Exemple de balise strong :

<strong>Texte important</strong>

style

La balise <style> est utilisée pour inclure des règles de style CSS directement dans un document HTML. Elle permet de définir le style visuel des éléments présents dans la page. Les règles de style spécifiées à l'intérieur de la balise <style> sont appliquées uniquement aux éléments du document courant.

Exemple de balise style :

<style>
          body {
            font-family: Arial, sans-serif;
            background-color: #f9f9f9;
            color: #333;
          }
        </style>

sub

La balise <sub> est utilisée pour représenter du texte en indice (en bas de ligne) dans un document HTML. Elle est souvent utilisée pour afficher des formules chimiques, des équations mathématiques, des notes de bas de page, etc.

Exemple de balise sub :

<sub>H</sub>2<sub>O</sub>

sup

La balise <sup> est utilisée pour représenter du texte en exposant (en haut de ligne) dans un document HTML. Elle est souvent utilisée pour afficher des indices, des exposants, des puissances, etc.

Exemple de balise sup :

10<sup>2</sup>

T

table, thead, tbody, tfoot, tr, th, td, colgroup, col, caption

Ces balises sont utilisées pour créer et structurer des tableaux de données dans un document HTML.

Exemple de balises de tableaux :

<table>
          <thead>
            <tr>
              <th>Entête 1</th>
              <th>Entête 2</th>
            </tr>
          </thead>
          <tbody>
            <tr>
              <td>Donnée 1</td>
              <td>Donnée 2</td>
            </tr>
          </tbody>
        </table>

time

La balise <time> est utilisée pour représenter une date ou une heure dans un document HTML. Elle peut être utilisée pour afficher des informations sur les horaires, les dates, les événements planifiés, etc., et facilite également l'accès et la manipulation de ces informations via des scripts.

Exemple de balise time :

<p>La date d'aujourd'hui est <time datetime="2023-06-02">2 juin 2023</time>.</p>

title

La balise `title` est utilisée pour spécifier le titre d'un document HTML. Le titre apparaît généralement dans la barre de titre du navigateur ou dans l'onglet de la page.

Exemple de balise `title` :

<head>
          <title>Titre de la page</title>
        </head>

Twig

Twig est un moteur de template pour le langage de programmation PHP. Il permet de séparer la logique métier de la présentation en utilisant des fichiers de template. Twig offre une syntaxe claire et concise pour générer des pages web dynamiques.

Exemple de code Twig :

{% raw %}
        <h1>{{ title }}</h1>
        <p>{{ content }}</p>
        {% endraw %}

U

u

La balise <u> est utilisée pour souligner un texte dans un document HTML. Elle peut être utilisée pour indiquer que le texte est une annotation, une correction, une entrée utilisateur ou toute autre information similaire.

Exemple de balise u :

<u>Texte souligné</u>

V

var

La balise <var> est utilisée pour représenter une variable dans un document HTML. Elle est souvent utilisée dans des contextes scientifiques ou mathématiques pour indiquer des variables dans des équations ou des formules.

Exemple de balise var :

<var>x</var>

W

wbr

La balise <wbr> est utilisée pour indiquer un point de coupure suggéré dans un mot lorsqu'il est nécessaire de le diviser en plusieurs lignes. Elle est souvent utilisée pour améliorer la lisibilité du texte lorsque la largeur de la fenêtre ou du conteneur change, en permettant au navigateur de choisir le meilleur endroit pour effectuer la coupure.

Exemple de balise wbr :

<p>Mot long <wbr>à couper</p>
