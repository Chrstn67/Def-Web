Def'Web

Vocabulaire

A

a

La balise <a> définit un lien hypertexte. Elle est utilisée pour créer des liens vers d'autres pages, des fichiers ou des emplacements dans la même page.

Exemple de balise <a> :

<a href="https://example.com">Lien vers exemple.com</a>

abbr

La balise <abbr> définit une abréviation ou un acronyme. Elle est utilisée pour indiquer la signification complète d'un mot ou d'une expression lorsqu'il est survolé par la souris.

Exemple de balise <abbr> :

<abbr title="Hypertext Markup Language">HTML</abbr>

address

La balise <address> définit les informations de contact pour l'auteur ou le propriétaire d'un document ou d'une section. Elle peut contenir une adresse postale, un numéro de téléphone, une adresse e-mail, etc.

Exemple de balise <address> :

<address>
          John Doe
          123 Rue des Exemples
          12345 Ville
        </address>

article

La balise <article> est utilisée pour représenter un contenu indépendant et autonome dans un document HTML. Elle peut être utilisée pour encapsuler des articles de blog, des commentaires, des widgets, etc. Elle devrait avoir un sens complet en étant extraite de son contexte.

Exemple de balise article :

<article>
          <h2>Titre de l'article</h2>
          <p>Contenu de l'article...</p>
        </article>

aside

La balise <aside> est utilisée pour représenter un contenu connexe ou supplémentaire qui est lié au contenu principal du document, mais qui peut être considéré comme distinct. Elle peut être utilisée pour afficher des encarts publicitaires, des citations, des biographies, etc.

Exemple de balise aside :

<aside>
          <h3>À propos de l'auteur</h3>
          <p>Informations supplémentaires...</p>
        </aside>

B

b

La balise <b> est utilisée pour mettre en évidence un texte important dans un document HTML. Par défaut, le texte entouré de balises <b> est affiché en gras, mais son utilisation actuelle est plus générale et dépend de la signification donnée au texte.

Exemple de balise b :

<b>Texte important</b>

background

La propriété CSS "background" est utilisée pour définir l'arrière-plan d'un élément. Elle regroupe plusieurs sous-propriétés qui permettent de définir la couleur, l'image, la position et d'autres paramètres de l'arrière-plan.

Exemple d'utilisation de la propriété "background" : 

.element {
          background: #f9f9f9 url("background.jpg") no-repeat center top;
        }

background-attachment

La propriété CSS "background-attachment" est utilisée pour définir si l'arrière-plan d'un élément défile avec le contenu ou reste fixe lorsque l'utilisateur fait défiler la page.

Exemple d'utilisation de la propriété "background-attachment" : 

.element {
          background-attachment: fixed;
        }

background-color

La propriété CSS "background-color" est utilisée pour définir la couleur de l'arrière-plan d'un élément.

Exemple d'utilisation de la propriété "background-color" : 

.element {
          background-color: #f9f9f9;
        }

background-image

La propriété CSS "background-image" est utilisée pour définir une image comme arrière-plan d'un élément.

Exemple d'utilisation de la propriété "background-image" : 

.element {
          background-image: url("background.jpg");
        }

background-position-x / background-position-y

Les propriétés CSS "background-position-x" et "background-position-y" sont utilisées pour définir la position horizontale et verticale de l'arrière-plan d'un élément, respectivement.

Exemple d'utilisation des propriétés "background-position-x" et "background-position-y" : 

.element {
          background-position-x: center;
          background-position-y: top;
        }

background-position

La propriété CSS "background-position" est utilisée pour définir la position horizontale et verticale de l'arrière-plan d'un élément en utilisant une seule valeur.

Exemple d'utilisation de la propriété "background-position" : 

.element {
          background-position: center top;
        }

Balise

En HTML, une balise est un élément utilisé pour marquer et structurer le contenu d'une page web. Elle est composée d'une balise ouvrante (<span>) et d'une balise fermante (</span>). Les balises définissent la structure et le sens du contenu.

Exemple de balise HTML :

<h1>Titre</h1>

base

La balise `base` définit l'URL de base pour les liens d'une page HTML. Elle est utilisée pour spécifier l'URL de base à partir de laquelle tous les liens relatifs dans le document seront résolus.

Exemple de balise `base` :

<head>
          <base href="https://www.example.com/">
        </head>

bdi

La balise <bdi> est utilisée pour isoler une partie de texte qui peut être formatée différemment en fonction de la direction du texte environnant. Elle est souvent utilisée pour gérer correctement l'affichage de texte dans des langues qui s'écrivent de droite à gauche (comme l'arabe ou l'hébreu) lorsqu'il est mélangé avec du texte de gauche à droite.

Exemple de balise bdi :

<bdi>العالم</bdi>

bdo

La balise <bdo> est utilisée pour spécifier explicitement la direction d'un texte dans un document HTML. Elle est souvent utilisée pour formater correctement le texte dans des langues qui s'écrivent de droite à gauche (comme l'arabe ou l'hébreu) ou de gauche à droite (comme l'anglais ou le français).

Exemple de balise bdo :

<bdo dir="rtl">العالم</bdo>

blockquote

La balise <blockquote> est utilisée pour citer un contenu provenant d'une autre source. Elle indique que le texte cité est une citation et peut être utilisée pour mettre en valeur ou styliser le texte cité.

Exemple de balise blockquote :

<blockquote>
          Voici une citation.
        </blockquote>

body

La balise `body` représente le contenu principal d'une page HTML. Elle contient tous les éléments visibles à l'écran, tels que le texte, les images, les liens, etc.

Exemple de balise `body` :

<body>
          <h1>Titre de la page</h1>
          <p>Contenu de la page...</p>
        </body>

border

La propriété CSS "border" est utilisée pour définir la bordure d'un élément. Elle permet de spécifier la largeur, le style et la couleur de la bordure.

Exemple d'utilisation de la propriété "border" : 

.element {
          border: 1px solid black;
        }

Exemple d'utilisation de la propriété "border-bottom" : 

.element {
          border-bottom: 2px dashed red;

border-bottom-color

Exemple d'utilisation de la propriété "border-bottom-color" :

Exemple d'utilisation de la propriété "border-bottom-style" :

          border-bottom-style: dotted;

border-bottom-width

La propriété CSS "border-bottom-width" est utilisée pour définir la largeur de la bordure inférieure d'un élément.

Exemple d'utilisation de la propriété "border-bottom-width" :

          border-bottom-width: 3px;
        }

border-collapse

La propriété CSS "border-collapse" est utilisée pour spécifier si les bordures des cellules d'un tableau doivent être fusionnées en une seule bordure ou séparées.

Exemple d'utilisation de la propriété "border-collapse" : 

table {
          border-collapse: collapse;
        }

border-color

La propriété CSS "border-color" est utilisée pour définir la couleur des bordures d'un élément. Elle peut prendre plusieurs valeurs pour spécifier différentes couleurs de bordure.

Exemple d'utilisation de la propriété "border-color" : 

.element {
          border-color: red green blue;

border-left

Exemple d'utilisation de la propriété "border-left-style" :

border-left-width

La propriété CSS "border-left-width" est utilisée pour définir la largeur de la bordure gauche d'un élément.

Exemple d'utilisation de la propriété "border-left-width" :

          border-left-width: 2px;

border-right

La propriété CSS "border-right" est utilisée pour définir la bordure droite d'un élément. Elle permet de spécifier la largeur, le style et la couleur de la bordure droite.

Exemple d'utilisation de la propriété "border-right" : 

.element {
          border-right: 1px solid black;
        }

border-right-color

La propriété CSS "border-right-color" est utilisée pour définir la couleur de la bordure droite d'un élément.

Exemple d'utilisation de la propriété "border-right-color" : 

.element {
          border-right-color: #00FF00;

border-right-style

La propriété CSS "border-right-style" est utilisée pour définir le style de la bordure droite d'un élément.

Exemple d'utilisation de la propriété "border-right-style" : 

.element {
          border-right-style: double;
        }

border-right-width

La propriété CSS "border-right-width" est utilisée pour définir la largeur de la bordure droite d'un élément.

Exemple d'utilisation de la propriété "border-right-width" : 

.element {
          border-right-width: 3px;
        }

border-spacing

La propriété CSS "border-spacing" est utilisée pour spécifier l'espace entre les cellules d'un tableau lorsque la propriété "border-collapse" est définie sur "separate".

Exemple d'utilisation de la propriété "border-spacing" : 

table {
          border-collapse: separate;
          border-spacing: 5px;
        }

border-style

La propriété CSS "border-style" est utilisée pour définir le style des bordures d'un élément.

Exemple d'utilisation de la propriété "border-style" : 

.element {
          border-style: dotted;
        }

border-top

La propriété CSS "border-top" est utilisée pour définir la bordure supérieure d'un élément. Elle permet de spécifier la largeur, le style et la couleur de la bordure supérieure.

border-top-style

        }

border-top-width

La propriété CSS "border-top-width" définit la largeur de la bordure supérieure d'un élément.

Exemple d'utilisation de la propriété "border-top-width" : 

.element {
          border-top-width: 2px;

border-width

La propriété CSS "border-width" définit la largeur des quatre bordures d'un élément dans l'ordre suivant : bordure supérieure, bordure droite, bordure inférieure, bordure gauche.

Exemple d'utilisation de la propriété "border-width" : 

.element {

bottom

La propriété CSS "bottom" spécifie la distance entre le bas d'un élément et le bord inférieur de son conteneur.

Exemple d'utilisation de "bottom" :

.element {
          position: absolute;

        }

br

La balise <br> est utilisée pour insérer un saut de ligne (retour à la ligne) à l'intérieur d'un paragraphe ou d'un texte. Elle est généralement utilisée lorsque vous souhaitez créer une nouvelle ligne sans créer un nouveau paragraphe.

Exemple de balise br :

Texte sur une ligne <br> Texte sur une nouvelle ligne

C

Cahier des charges

Un cahier des charges est un document qui décrit de manière détaillée les spécifications, les objectifs, les contraintes et les attentes d'un projet. Il sert de référence commune entre le client et l'équipe de développement pour définir les fonctionnalités, les délais, les ressources nécessaires et les critères de réussite du projet.

Exemple de contenu d'un cahier des charges : 

Objectif du projet :
        - Créer un site web e-commerce pour la vente de produits électroniques.
        Fonctionnalités :
        
        Page d'accueil avec mise en avant des produits populaires.
        Catalogue de produits avec filtres de recherche.
        Panier d'achat et processus de paiement.
        Gestion des utilisateurs (inscription, connexion, profil).
        Système de commentaires et d'évaluations des produits.
        Contraintes :
        
        Utiliser le framework PHP Laravel.
        Respecter les bonnes pratiques de sécurité.

caption-side

La propriété CSS "caption-side" est utilisée pour spécifier l'alignement de la légende d'un tableau par rapport à ce dernier.

Exemple d'utilisation de la propriété "caption-side" : 

table {
          caption-side: bottom;
        }

cite

La balise <cite> est utilisée pour indiquer une référence à une source dans un document HTML. Elle est généralement utilisée pour citer des livres, des articles, des œuvres d'art, etc., et peut être stylisée pour indiquer visuellement qu'il s'agit d'une citation.

Exemple de balise cite :

<cite>Titre de l'article</cite>

clear

La propriété CSS "clear" spécifie si un élément doit être déplacé en dessous des éléments flottants voisins qui le précèdent.

Exemple d'utilisation de "clear" :

.element {
          clear: left;
        }

clip

La propriété CSS "clip" spécifie une région rectangulaire à afficher d'un élément positionné de façon absolue.

Exemple d'utilisation de "clip" :

.element {
          position: absolute;
          clip: rect(0px, 100px, 100px, 0px);
        }

code

La balise <code> est utilisée pour représenter du code informatique dans un document HTML. Elle est généralement utilisée pour afficher du code source, des commandes, des expressions, etc., de manière à ce qu'elles soient distinguées du texte normal et à largeur fixe.

Exemple de balise code :

<code>var x = 5;</code>

color

La propriété CSS "color" est utilisée pour définir la couleur du texte à l'intérieur d'un élément HTML. Elle permet de spécifier la couleur du texte en utilisant différents formats tels que les noms de couleurs prédéfinis, les valeurs hexadécimales, les valeurs RVB ou les valeurs HSL.

Exemple d'utilisation de la propriété "color" : 

p {
          color: red;
        }

Composant

Un composant est une unité autonome et réutilisable de code qui encapsule des fonctionnalités spécifiques et peut être utilisé pour construire des interfaces utilisateur. Les composants sont largement utilisés dans les frameworks de développement web tels que React, Angular et Vue.js.

Exemple de composant :

// Exemple de composant en React
        class Button extends React.Component {
          render() {
            return (
              <button onClick={this.props.onClick}>
                {this.props.label}
              </button>
            );
          }
        }

content

La propriété CSS "content" est utilisée pour insérer du contenu généré dans un élément à l'aide des pseudo-éléments "::before" et "::after". Elle permet d'ajouter du texte, des images ou d'autres éléments de contenu à l'élément sélectionné.

Exemple d'utilisation de la propriété "content" : 

p::before {
          content: "Avant ";
        }
        p::after {
        content: " Après";
        }

counter-increment

La propriété CSS "counter-increment" est utilisée pour incrémenter la valeur d'un compteur spécifié. Elle est couramment utilisée en combinaison avec les pseudo-éléments "::before" et "::after" pour créer des listes numérotées ou d'autres éléments de numérotation.

Exemple d'utilisation de la propriété "counter-increment" : 

ol {
          counter-reset: section;
        }
        li::before {
        counter-increment: section;
        content: "Section " counter(section) ": ";
        }

counter-reset

La propriété CSS "counter-reset" est utilisée pour réinitialiser la valeur d'un compteur spécifié. Elle est couramment utilisée en combinaison avec les pseudo-éléments "::before" et "::after" pour créer des listes numérotées ou d'autres éléments de numérotation.

Exemple d'utilisation de la propriété "counter-reset" : 

ol {
          counter-reset: section;
        }
        li::before {
        counter-increment: section;
        content: "Section " counter(section) ": ";
        }

cursor

La propriété CSS "cursor" est utilisée pour spécifier le type de curseur qui doit être affiché lorsque l'utilisateur survole un élément.

Exemple d'utilisation de la propriété "cursor" : 

.element {
          cursor: pointer;
        }

D

data

La balise <data> est utilisée pour représenter une valeur numérique ou scientifique dans un document HTML. Elle peut être utilisée pour ajouter des métadonnées aux éléments HTML et pour faciliter la manipulation des données via des scripts.

Exemple de balise data :

<p>La population mondiale est d'environ <data value="7800000000"></data> habitants.</p>

del

La balise <del> est utilisée pour représenter du texte supprimé ou barré dans un document HTML. Elle est souvent utilisée pour indiquer des suppressions, des omissions ou des modifications dans le texte existant, notamment dans le contexte de la comparaison ou de la révision de texte.

Exemple de balise del :

<del>Texte supprimé</del>

details, summary, dialog

Ces balises sont utilisées pour créer des éléments interactifs de l'interface utilisateur, tels que des sections déroulantes (avec les balises details et summary) et des boîtes de dialogue modales (avec la balise dialog).

Exemple de balises interactives :

<details>
          <summary>Titre de la section</summary>
          Contenu de la section
        </details>
        
        <dialog open>
          <p>Contenu de la boîte de dialogue</p>
          <button>Fermer</button>
        </dialog>

dfn

La balise <dfn> est utilisée pour indiquer la définition d'un terme dans un document HTML. Elle est souvent utilisée pour fournir une explication ou un contexte supplémentaire pour un terme qui peut ne pas être familier à tous les lecteurs.

Exemple de balise dfn :

<dfn>Terme à définir</dfn>

direction

La propriété CSS "direction" est utilisée pour spécifier la direction de l'écriture du texte dans un élément. Elle détermine si le texte est écrit de gauche à droite (valeur par défaut) ou de droite à gauche pour les langues telles que l'arabe ou l'hébreu.

Exemple d'utilisation de la propriété "direction" : 

.arabic-text {
          direction: rtl;
        }

display

La propriété CSS "display" spécifie le type d'affichage utilisé pour un élément.

Exemple d'utilisation de "display" :

.element {
          display: flex;
        }

div

La balise <div> est utilisée pour créer une division ou une section générique dans un document HTML. Elle est souvent utilisée comme un conteneur pour regrouper d'autres éléments HTML et appliquer des styles ou des fonctionnalités à ces éléments groupés.

Exemple de balise div :

<div>
          <p>Contenu du div</p>
          <img src="image.jpg" alt="Image">
        </div>

dl, dt, dd

Les balises <dl>, <dt> et <dd> sont utilisées pour créer une liste de définitions dans un document HTML. <dl> est le conteneur principal, <dt> est utilisée pour définir le terme ou le nom, et <dd> est utilisée pour définir la description ou la signification du terme.

Exemple de balises dl, dt, dd :

<dl>
          <dt>Terme 1</dt>
          <dd>Description 1</dd>
          <dt>Terme 2</dt>
          <dd>Description 2</dd>
        </dl>

Doctype

Le doctype est une déclaration placée au début d'un document HTML pour indiquer au navigateur quel type de document il est en train de traiter. Il définit également la version de HTML utilisée et permet au navigateur d'interpréter correctement le code HTML.

Exemple de doctype HTML5 :

<!DOCTYPE html>

E

Élément

En HTML, un élément est une partie de la structure d'une page web qui est délimitée par une balise ouvrante et une balise fermante, ou une balise auto-fermante. Les éléments peuvent contenir du texte, des images, d'autres éléments et des attributs qui définissent leurs caractéristiques.

Exemple d'élément HTML :

<p>Ceci est un paragraphe.</p>

empty-cells

La propriété CSS "empty-cells" est utilisée pour spécifier la manière dont les cellules vides d'un tableau doivent être affichées.

Exemple d'utilisation de la propriété "empty-cells" : 

table {
  empty-cells: hide;
}

F

figure, figcaption

Les balises <figure> et <figcaption> sont utilisées pour encapsuler une image ou un média avec une légende associée dans un document HTML. La balise <figure> est utilisée comme conteneur principal et la balise <figcaption> est utilisée pour définir la légende.

Exemple de balises figure, figcaption :

<figure>
          <img src="image.jpg" alt="Image">
          <figcaption>Légende de l'image</figcaption>
        </figure>

filter

La propriété CSS "filter" est utilisée pour appliquer des effets visuels, tels que des filtres de couleur, de flou ou de luminosité, à un élément.

Exemple d'utilisation de la propriété "filter" : 

.element {
          filter: grayscale(100%);
        }

flexbox

Flexbox est une méthode de disposition flexible en CSS qui permet de créer des mises en page réactives et adaptatives. Il utilise un système de conteneurs (éléments parent) et d'éléments enfants pour créer des designs flexibles et dynamiques.

Exemple d'utilisation de flexbox :

.container {
        display: flex;
        justify-content: center;
        align-items: center;
      }
      .item {
      flex: 1;
      }

float

La propriété CSS "float" spécifie que l'élément doit être placé le plus à gauche ou le plus à droite possible, et que le texte et les autres éléments doivent s'écouler autour de lui.

Exemple d'utilisation de "float" :

.element {
          float: left;
        }

font

La propriété CSS "font" est une propriété raccourcie qui permet de définir plusieurs propriétés de police en une seule déclaration. Elle est utilisée pour spécifier la famille de police, la taille de police, le style de police et d'autres propriétés liées à la police.

Exemple d'utilisation de la propriété "font" : 

.my-text {
        font: 16px/1.5 Arial, sans-serif;
      }

font-family

La propriété CSS "font-family" est utilisée pour spécifier la famille de polices utilisée pour afficher le texte dans un élément. Elle permet de définir plusieurs noms de polices préférées, classées par ordre de priorité, afin que le navigateur puisse choisir la première police disponible sur le système de l'utilisateur.

Exemple d'utilisation de la propriété "font-family" : 

.my-text {
        font-family: Arial, sans-serif;
      }

font-size

La propriété CSS "font-size" est utilisée pour spécifier la taille de la police utilisée pour afficher le texte dans un élément. Elle permet de définir la taille de police en pixels, en pourcentage, en em ou en d'autres unités de mesure.

Exemple d'utilisation de la propriété "font-size" : 

.my-text {
        font-size: 16px;
      }

font-size-adjust

La propriété CSS "font-size-adjust" est utilisée pour ajuster la taille de la police en fonction de la hauteur x des lettres majuscules dans une police de substitution, afin de maintenir la cohérence de l'apparence du texte lors du changement de police.

Exemple d'utilisation de la propriété "font-size-adjust" : 

.my-text {
        font-size-adjust: 0.5;
      }

font-stretch

La propriété CSS "font-stretch" est utilisée pour spécifier l'étirement ou la compression horizontal de la largeur des caractères dans une police de caractères. Elle permet de contrôler la largeur relative de la police, en utilisant des valeurs prédéfinies ou des pourcentages personnalisés.

Exemple d'utilisation de la propriété "font-stretch" : 

.my-text {
        font-stretch: expanded;
      }

En programmation, un reducer est une fonction qui prend en entrée un état courant et une action, et qui retourne un nouvel état résultant de l'application de l'action à l'état courant. Les reducers sont couramment utilisés dans les architectures de gestion d'état, comme Redux, pour mettre à jour et gérer l'état de l'application de manière prévisible et immuable.

Exemple d'utilisation de reducer avec Redux : 

const initialState = {
        count: 0
      };
      function counterReducer(state = initialState, action) {
      switch (action.type) {
      case 'INCREMENT':
      return { ...state, count: state.count + 1 };
      case 'DECREMENT':
      return { ...state, count: state.count - 1 };
      default:
      return state;
      }
      }

          dispatch({ type: 'INCREMENT' });
        };

          dispatch({ type: 'DECREMENT' });
        };
